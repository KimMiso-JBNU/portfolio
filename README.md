# Bojagi Portfolio

GitHub Pages 배포용 정적 포트폴리오입니다.

## 배포 방법

1. GitHub에서 새 공개 저장소를 만듭니다.
2. 이 폴더의 파일 전체를 저장소에 업로드합니다.
3. 저장소의 `Settings > Pages`로 이동합니다.
4. `Build and deployment`에서 `Source`를 `Deploy from a branch`로 선택합니다.
5. `Branch`를 `main` / `/ (root)`로 선택한 뒤 저장합니다.
6. 잠시 후 `https://사용자이름.github.io/저장소이름/` 주소로 접속합니다.

## 포함 파일

- `index.html`: 메인 페이지
- `about.html`: About 페이지
- `style.css`: 공통 스타일
- `script.js`: 인터랙션 스크립트

## 참고

저장소 이름을 `username.github.io` 형태로 만들면 주소가 루트 도메인으로 열립니다.
